# Prada Clone / Spark Stage Commerce Prototype

Private learning project for studying a Prada-style storefront that will be reshaped into a Spark Stage fashion commerce skripsi prototype.

## Structure

- `frontend/` contains the Vite frontend app, current static clone pages, React/TypeScript migration baseline, frontend tests, and visual artifacts.
- `supabase/` stays at the repo root for Supabase config, migrations, and Edge Functions.
- `docs/` stays at the repo root for planning and skripsi context.
- Root `package.json` uses npm workspaces as a light monorepo entrypoint while the frontend app remains in `frontend/`.

## Commands

From the repo root:

```bash
npm run dev
npm run build
npm run typecheck
npm run preview
npm run test:e2e
```

Direct frontend commands still work:

```bash
cd frontend
npm install
npm run dev
npm run build
npm run typecheck
npm run preview
npm run test:e2e
```

## Generated Artifacts

- `frontend/dist/` is generated by `npm run build`.
- `frontend/test-results/` and `frontend/playwright-report/` are generated by Playwright.
- `frontend/.playwright-mcp/` is generated by MCP browser sessions.
- `frontend/public/assets/harvested/` is reserved for raw captured reference assets and is ignored by git by default.

## Project Notes

The current clone still has static HTML pages while the agreed skripsi stack is React + TypeScript + Vite + Supabase + DOKU Checkout sandbox. The React baseline is in `frontend/src/`; existing static pages remain as the visual reference while we migrate page-by-page.
